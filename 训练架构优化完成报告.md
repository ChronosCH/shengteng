## 🚀 使用方式

### 方式一：交互式启动（推荐）
```bash

# 然后按提示选择训练器
```

### 方式二：直接训练
```bash
cd training
python train.py --model optimal     # 最优训练器
python train.py --model enhanced    # 增强训练器
```

### 方式三：自定义参数
```bash
cd training
python train.py --model optimal --epochs 100 --batch-size 4
```
